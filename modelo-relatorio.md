# RELATÓRIO DE IMPLEMENTAÇÃO DE SERVIÇOS AWS

Data: 25/08;2023
Empresa: Grupo Pacheco 
Responsável: Alan Araújo

## Introdução
Este relatório apresenta o processo de implementação de ferramentas na empresa Grupo Pacheco, realizado por Alan Araujo. O objetivo do projeto foi elencar 3 serviços AWS, com a finalidade de realizar diminuição de custos imediatos.

## Descrição do Projeto
O projeto de implementação de ferramentas foi dividido em 3 etapas, cada uma com seus objetivos específicos. A seguir, serão descritas as etapas do projeto:

Etapa 1: 
- Amazon EC2 
- Maquinas Virtuais
- Trocar servidores fisicos de Aplicação e banco de dados que tem no data center por servidores Virtuais. Assim evitando custo de manutenções em servidores fisicos.

Etapa 2: 
- Amazon RDS
- Banco de Dados
- A ferramenta irá servir para eliminar tarefas administrativas de banco de dados ineficientes e onerosas sem a necessidade de provisionamento de infraestrutura ou manutenção de software. 

Etapa 3: 
- Amazon CloudWatch
- Monitoramento
- Essa ferramenta irá monitorar toda a infraestrutura e aplicação para verificar metricas e comportamentos. 



## Conclusão
A implementação das ferraments apresentadas irá trazer economia, disponibilidade e controle sobre a infraestrutura e aplicações.
